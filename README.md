### 題目

1. 資料源使用免費的API - https://restcountries.eu/
   
2. 需求

   - 目錄必須顯示欄位,()内為Api資料物件的屬性
    
     - 國旗(圖片資源請使用 flag)
       
     - 國家名稱（name）
       
     - 2位國家代碼(alpha2Code)
       
     - 3位國家代碼(alpha3Code)
       
     - 母語名稱(nativeName)
       
     - 替代國家名稱(altSpellings)
       
     - 國際電話區號(callingCodes)
    
   - 以國名搜尋功能(模糊搜尋)
     
   - 以國名排序功能（正序、倒序）
     
   - 分頁(每頁25筆)
     
   - 點擊國家名稱后顯示一個Modal，裏面顯示該國家的其他資訊。

3. 服務必須部署到GitHub Pages，並保留完整的提交記錄。
   
4. 若能使用任一前端框架(Vue/React/Angular)更好！
